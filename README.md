# affirmeep
Express JS demo API
Two endpoints available: 
/  -- Returns Hello World 
/affirmations -- Gives you a random affirmation

## Quick Start

  Install dependencies:

```bash
$ npm install
```

  Start the server:

```bash
$ npm start
```

  View the website at: http://localhost:3000

## Tests

  To run the test suite, first install the dependencies, then run `npm test`:

```bash
$ npm install
$ npm test
```
